Themantic for MODx Revolution
================================

Version: 1.0.0-beta
Release Date: 2025

Overview
--------
Themantic is the most versatile MODx front-end template that seamlessly 
integrates the power of MODx Revolution with the beauty and flexibility 
of Semantic/Fomantic UI.

Features
--------
- Semantic/Fomantic UI 2.5+ Integration
- Responsive Mobile-First Design
- MODx Manager Integration
- Pre-built Templates (Landing, Blog, E-commerce, Corporate)
- Custom Snippets and Plugins
- Easy Customization
- Production-Ready

Installation
------------
See INSTALLATION.md for detailed installation instructions.

Usage
-----
After installation, select a Themantic template when creating new resources
in the MODx Manager. Customize the appearance using Semantic/Fomantic UI components
and MODx template variables.
